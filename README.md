# This is the repo for the backend of Devnotes app

## Technologies Used

- Express JS
- Mongoose
- Typescript

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Run your unit tests

```
npm run test
```

### Lints files

```
npm run tslint
```

### Fixes lint errors

```
npm run tslint:fix
```
